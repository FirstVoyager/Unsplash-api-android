# Unsplash-api-android
Library for android

## Add to android projects

Add it in your root build.gradle at the end of repositories:

```bash
allprojects {
	repositories {
		maven { url 'https://jitpack.io' }
	}
}
```

Add the dependency

```bash
dependencies {
	implementation 'com.github.FirstVoyager:Unsplash-api-android:Tag'
}
```
